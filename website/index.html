Engineered for Power Users

A modular system designed to bypass cognitive overhead, automate desktop environments, and run offline.

Dual-Brain Resilience

Integrates a structured Gemini-powered planner (Planner V2) with direct REST fallbacks and local Ollama model options (gemma3:4b/llama3.2). Never fails when offline.

🎙️

Natural Speech Pipeline

Features hands-free wake word recognition ("Vortex") with noise isolation, natural neural Edge TTS synthesis, and barge-in voice command cancellation.

🛠️

Deep System Hooks

Direct PowerShell start menu indexing caches 100% of installed executables and store apps. Includes Pycaw hardware endpoints for direct volume scaling.

👁️

Screen & OCR Awareness

Takes high-resolution active buffers using PyAutoGUI and parses screen contents via Tesseract OCR to diagnose compile errors or stack traces dynamically.

🎯

Multi-Step Macros

Executes workflow actions using triggers. Set custom modes like Interview Preparation, DSA Study, Gaming, or Productivity with single statements.

📊

Observability & RAG

Logs millisecond tool execution latencies to SQLite, parsed via a Streamlit analytics panel. Recalls past context semantically using ChromaDB vector embeddings.
